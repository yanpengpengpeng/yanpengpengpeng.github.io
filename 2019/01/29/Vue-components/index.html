组件 - components

  • 定义: 组件的出现，就是为了拆分vue实例的代码量的，能够让我们以不同的组件，来划分不同的功能模块，将来我们需要什么样的功能，就可以去调用对应的组件即可;
  • 组件化和模块化的区别
    • 模块化:是从代码逻辑的角度进行划分的,方便代码分层开发,保证每个功能模块的只能单一。
    • 组件化:是从UI界面的角度进行划分的，前端的组件化，方便UI组件的重用。
